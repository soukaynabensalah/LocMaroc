# Locato

A simple client-server web application.

## Structure

- **client/**: Contains the frontend code (HTML, CSS, JS).
- **server/**: Contains the backend code (Node.js/Express).

## Getting Started

1.  Navigate to the `server` directory:
    ```bash
    cd server
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the server:
    ```bash
    npm start
    ```
4.  Open your browser and visit `http://localhost:5000`.
